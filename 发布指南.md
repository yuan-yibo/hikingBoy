#### 1️⃣ 注册小程序账号
1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册" → 选择"小程序"
3. 填写账号信息（使用未注册过的邮箱）
4. 选择主体类型：
   - **个人**：适合个人使用和分享给朋友
   - **企业/组织**：需要营业执照等资质
5. 完成注册后，进入"设置" → "开发设置"，复制 **AppID**

#### 2️⃣ 更新项目配置
将你的 AppID 更新到以下文件：
- `project.config.json` 第 23 行，替换 `"appid": "wx3916326d14481062"`

#### 3️⃣ 准备素材资源

**必需图标（需自行准备）：**
- 小程序图标：1024×1024px PNG格式
- TabBar图标（6张）：81×81px PNG格式
  - `images/record.png` - 记录（未选中）
  - `images/record-active.png` - 记录（选中）
  - `images/stats.png` - 统计（未选中）
  - `images/stats-active.png` - 统计（选中）
  - `images/settings.png` - 设置（未选中）
  - `images/settings-active.png` - 设置（选中）

**建议准备：**
- 启动页图片：750×1334px
- 宣传图：至少5张小程序功能截图

#### 4️⃣ 开发工具配置

1. 下载微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 安装并登录（使用绑定了小程序的微信扫码登录）
3. 导入项目：
   - 选择项目目录：`E:\workspace\wechatTest`
   - AppID：填写你注册的小程序 AppID
   - 项目名称：亲子爬山记

#### 5️⃣ 代码调试与优化

**在开发者工具中检查：**
- [ ] 所有页面正常显示
- [ ] 添加/编辑/删除记录功能正常
- [ ] 主题切换生效
- [ ] 统计数据准确
- [ ] 分享文案生成正确
- [ ] 图片上传和预览正常

**性能优化建议：**
- [ ] 检查代码包大小（建议 < 2MB）
- [ ] 图片压缩优化
- [ ] 清理未使用的文件

#### 6️⃣ 上传代码

1. 在开发者工具中点击"上传"按钮
2. 填写版本号：`1.0.0`
3. 填写项目备注：`初始版本发布`
4. 点击"上传"

#### 7️⃣ 提交审核

1. 登录微信公众平台：https://mp.weixin.qq.com/
2. 进入"管理" → "版本管理"
3. 找到刚上传的开发版本，点击"提交审核"
4. 填写审核信息：
   - **功能页面**：记录、统计、设置
   - **服务类目**：生活服务 → 综合生活服务
   - **标签**：亲子、运动、记录
5. 上传测试账号（非必需）
6. 提交审核

#### 8️⃣ 发布上线

1. 审核通过后（通常1-7天）
2. 在"版本管理"中找到"审核通过"的版本
3. 点击"提交发布"
4. 发布成功！🎉

## 🔧 常见问题

### Q1: 审核被拒怎么办？
**A:** 查看拒绝原因，常见问题：
- 缺少隐私政策说明
- 功能描述不清晰
- 缺少必要的用户协议
- 需要补充资质证明

### Q2: TabBar图标在哪里准备？
**A:** 你需要自己设计或下载6张图标，放到 `images/` 目录，然后修改 `app.json` 中的 `tabBar` 配置，取消注释图标路径。

### Q3: 个人注册的小程序有限制吗？
**A:** 个人小程序有以下限制：
- 无法开通微信支付
- 无法使用某些敏感接口
- 可用服务类目较少
- 但记录类小程序完全够用！

### Q4: 如何分享给朋友使用？
**A:** 发布后，通过以下方式：
- 分享小程序码（在"设置"→"基本设置"中下载）
- 直接发送小程序卡片
- 发送小程序链接

### Q5: 数据会不会丢失？
**A:** 当前版本使用本地存储，数据保存在用户手机上：
- ✅ 优点：隐私安全，不需要服务器
- ⚠️ 注意：卸载小程序或清除缓存会丢失数据
- 💡 建议：定期使用分享功能备份重要记录
